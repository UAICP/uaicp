# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

- Be respectful and constructive.
- Assume good intent and focus on technical substance.
- Avoid harassment, personal attacks, and discriminatory language.

## Reporting

For conduct concerns, open a private maintainer contact through GitHub maintainers.

## Enforcement

Project maintainers may remove or reject contributions that violate this Code of Conduct.
