# UAICP Roadmap

This roadmap tracks post-`v0.3` enhancements. It is issue-driven and does not use timeline/date commitments.

## Complete

- baseline state model and invariant set
- baseline schema contracts
- baseline compliance categories
- initial adapter guidance
- initial reference implementation primitives

## Post-v0.3 Active Work

- normative wording hardening for `v0.3`
- machine-readable conformance harness contract
- adapter conformance walkthrough artifacts

## Post-v0.3 Backlog

- external conformance report examples
- reliability maturity profile proposal (`L1/L2/L3`)
- expanded adapter fixtures and migration guides

## Tracking Model

Roadmap execution is tracked in GitHub issues:

- [Specification Issues](https://github.com/UAICP/uaicp/issues)

Recommended labels:

- `roadmap`
- `workstream/spec`
- `workstream/schema`
- `workstream/conformance`
- `workstream/adapters`
- `workstream/docs`

All roadmap work should be represented in issues and linked from PRs.
